
What is iXlsxWriter.AspNet?
===========================

iXlsxWriter.AspNet, extends iXlsxWriter to work in AspNet (FullFramework) projects, contains extension methods to download XlsxInput instances 
as well as OutputResult, facilitating its use in this environment.
